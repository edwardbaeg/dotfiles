Keymaps:

### Display
- `.`: toggle showing hidden
- `,`: start sort options, eg, sort by modified time, `,M`

### Operations
- `space`: select
- `a`: create file or directory (end with `/`)
- `x`: yank (cut)
- `y`: yank (copy)
- `p`: paste
- `c`: start copy
  - `cf`: copy file name

### Help
- `~` show searchable help
